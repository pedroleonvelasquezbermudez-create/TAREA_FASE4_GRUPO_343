# Software FJ - Sistema de Gestión Integral

## Descripción
Este proyecto es una aplicación orientada a objetos desarrollada para gestionar clientes, servicios y reservas.

## Estructura del Proyecto
- `/src`: Contiene el código fuente del sistema.
- `registro_eventos.log`: (Local) Archivo donde se registran los errores y eventos.
